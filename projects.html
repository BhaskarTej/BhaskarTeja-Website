Driver Drowsiness Detection

Developed a real-time drowsiness detection system using machine learning techniques to monitor driver alertness and prevent accidents.

We implemented a convolutional neural network (CNN) for facial landmark detection to identify signs of drowsiness such as eye closure and head tilting.

We utilized OpenCV for video processing and image recognition tasks, integrating with the ML model to analyze live video feeds.
